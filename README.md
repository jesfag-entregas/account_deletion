# Página de Exclusão de Conta - Jesfag Entregas

Este repositório contém a página HTML `account_deletion.html` utilizada para atender à política de exclusão de conta exigida pelo Google Play.

## Finalidade

A página informa aos usuários do aplicativo **Jesfag Entregas** como solicitar a exclusão de sua conta, detalhando:

- O procedimento para solicitar a exclusão (via e-mail)
- O que acontece com os dados pessoais após a solicitação
- O prazo para remoção dos dados
- O contato oficial para requisições

## Como usar

1. **Hospede o arquivo `account_deletion.html`** em um servidor web acessível publicamente (ex: GitHub Pages, seu próprio domínio, etc).
2. **Adicione o link** para esta página na seção "Detalhes do app" no Google Play Console, conforme exigido pela política de exclusão de conta.

## Estrutura da página

- Exibe a logo da Jesfag Entregas
- Passos claros para o usuário solicitar a exclusão
- Lista de dados excluídos, mantidos e prazo de armazenamento
- E-mail de contato: jesfagentregas17@gmail.com

## Observações

- Certifique-se de manter o conteúdo atualizado conforme a política de privacidade do app.
- O e-mail de contato deve estar ativo para receber solicitações dos usuários.
